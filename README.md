# WebRTC 연구실 


### 목표

	* WebRTC를 보다 쉽게 이해하고 사용할 수 있도록 주요 API를 예제로 설명한다.
	* WebRTC를 활용할 수 있도록 흥미로운 응용 주제를 소개한다.

### 구성

	1. WebRTC 소개
	2. 마이크 & 캠 접근하기
	3. 비디오에 필터 적용하기
	4. 비디오를 이미지로 캡쳐하기
	5. 1:1 화상회의
	6. 화면 공유
	7. 음성 인식
	8. 파일 & 데이터 전송하기
	9. 녹화 하기
	10. 얼굴 인식
	11. 더 자세한 설명
