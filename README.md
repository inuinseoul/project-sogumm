## Project Sogumm

# 실행방법
1. node.js 운영체제 버전에 맞게 설치 http://www.nodejs.org
2. cmd(맥일 경우 터미널)로 다운로드 경로 이동 
3. npm install 명령어 실행
4. npm start 명령어 실행
5. 크롬브라우저에서 localhost:3000으로 접속

# 활용기술

- node.js<br>
- webRTC<br>
- socket.io<br>
- GoogleAPI<br>
- WebAudioAPI<br>
